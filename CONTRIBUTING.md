# Contributing to Quran Kol Youm ü§ù

Thank you for your interest in contributing to this project! Since this is a Sadaqah Jariyah project, your help is highly appreciated.

## How can you help?

### 1. Adding New Duaas
We want to keep the captions fresh. You can add more supplications to the list:
1. Open `duaa.json`.
2. Add your new Duaa to the list following the JSON format.
3. Submit a Pull Request.

### 2. Reporting Bugs
If you notice the bot stopped working or posted something incorrect, please open an **Issue** on GitHub.

### 3. Improving Documentation
If you think the instructions are not clear for non-technical users, feel free to update the `README.md` or `FACEBOOK_SETUP.md`.

### 4. Contact Us
If you need any help or have any questions, feel free to contact us 
- **Telegram:** [@Avax43](https://t.me/Avax43)

## Pull Request Process
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

May Allah reward you for your efforts!
